Informática para Internet

Sobre o Curso

O curso técnico em Informática para Internet integrado ao ensino médio tem duração de 3 anos e combina as disciplinas regulares do ensino médio com formação técnica em desenvolvimento web, preparando os alunos para atuarem no mercado de tecnologia.

O que você vai estudar

  • Programação Web (HTML, CSS, JavaScript)
  • Design de Interfaces e UX
  • Banco de Dados e Back-end
  • Desenvolvimento Mobile
  • Redes e Infraestrutura
  • Projetos Integradores
